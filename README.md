# 🎯 Decision Maker - by KEKO 🪴

> Stop overthinking. Let the game decide.

---

## ✨ What is it?

A fun web app that helps you make decisions using **mini-games** instead of thinking too much.

🎮 Enter a dilemma → play a game → get your answer

---

## 🎮 Games

* 🪙 **Coin Toss** → 3D animated flip
* 👯 **Friends Game** → fast questions + surprise dilemma
* 🎡 **Spinner** → spin the wheel
* ⚡ **Reaction** → click fast
* 🎲 **Dice** → odd / even decides

🚧 More games coming soon...
